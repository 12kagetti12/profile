# Satoshi Kageyama — Portfolio

> フルスタックエンジニアのポートフォリオサイトのソースコード

---

## サイト概要

| 項目 | 内容 |
|------|------|
| 技術スタック | Next.js 13 / TypeScript / Tailwind CSS |
| 構成 | 静的サイト生成（SSG）・5ページ構成 |
| 公開URL | [https://kageyama-portfolio.jp/](https://kageyama-portfolio.jp/) |

---

## ページ構成

| ページ | パス | 内容 |
|--------|------|------|
| トップ | `/` | 自己紹介・スキル概要 |
| Web制作 | `/web` | Web制作・開発の実績一覧 |
| 紙媒体 | `/paper` | 印刷物・紙媒体の制作実績 |

## ローカル起動

```bash
npm install
npm run dev
```

ブラウザで [https://localhost:3000](https://localhost:3000) を開く。

---

## ビルド・エクスポート

```bash
npm run build
```

静的ファイルが `out/` に生成されます。

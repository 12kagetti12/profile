# Installation

## Framework

```
npx create-next-app@latest
```

## prettier-plugin-tailwindcss

```
npm install -D prettier prettier-plugin-tailwindcss
```

Then add the plugin to your Prettier config

```
// prettier.config.js
module.exports = {
  plugins: ['prettier-plugin-tailwindcss'],
}
```
